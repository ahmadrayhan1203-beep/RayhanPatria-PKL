42. Perbedaan shared hosting & dedicated hosting

Shared hosting digunakan bersama banyak user, biaya murah, akses terbatas.
Dedicated hosting hanya satu user, server penuh, akses root, performa tinggi.

43. Alasan VPS lebih manageable

Akses root penuh, stabil, fleksibel, bisa pasang software sendiri.

44. Hosting untuk e-commerce perusahaan

Dedicated hosting atau cloud enterprise. Alasan: keamanan tinggi, performa stabil, integrasi keuangan & perbankan.

45. Saran untuk siswa membuat web pribadi

Menggunakan shared hosting, domain murah/gratis, memakai CMS seperti WordPress, isi dengan portofolio dan biodata.
