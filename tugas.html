41. Skema jaringan penyewaan hosting
Gunakan jaringan 10.10.1.0/24. IP server: 10.10.1.1. Klien diberikan range 10 IP misalnya 10.10.1.10–10.10.1.20. Setiap klien memakai VPS/virtual host dengan IP statis.

42. Perbedaan shared hosting dan dedicated hosting
Shared: server dipakai bersama, murah, akses terbatas.
Dedicated: satu server untuk satu pengguna, akses penuh, performa tinggi.

43. Alasan VPS lebih manageable
Akses root penuh, resource stabil, bisa install software sendiri, skalabilitas baik.

44. Hosting untuk e-commerce perusahaan
Dedicated Hosting / Cloud Enterprise. Alasannya: keamanan tinggi, performa stabil, integrasi sistem keuangan, traffic besar.

45. Saran untuk siswa yang ingin membuat web pribadi
Gunakan shared hosting, domain gratis (.tk), template sederhana, isi biodata, portofolio, dan catatan pelajaran.
