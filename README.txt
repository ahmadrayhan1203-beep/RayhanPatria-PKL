PKL Ahmad Rayhan - Static website (GitHub Pages)

Cara pakai:
1. Upload seluruh folder isi zip ke repository GitHub (pastikan file di root repository, bukan di dalam folder lain).
2. Aktifkan GitHub Pages via Settings -> Pages -> pilih branch main dan folder root (/).
3. Akses: https://<username>.github.io/<repo>

Struktur file:
- index.html
- pkl.html
- styles.css
- script.js
- assets/rayhan-pkl.jpg
- assets/rayhan-tower.jpg
